  • SPI_BUSY [read-only]: SPI Busy
    read 1: spi busy.
    read 0: spi idle
  • PLL_STATUS [read-only]: CDCE72010 PLL Status
    read 1: PLL locked.
    read 0: PLL not locked
  • ADC_CLK_LOCKED [read-only]: FPGA ADC clock locked
    read 1: FPGA ADC PLL locked.
    read 0: FPGA ADC PLL not locked
  • FMC_PRST [read-only]: FMC present
    read 1: FMC present.
    read 0: FMC not present
