3.1. Control register

HW prefix: acq_core_ctl
HW address: 0x0
C prefix: CTL
C offset: 0x0

7 6 5 4 3 2 1 0
RESERVED1[5:0] FSM_STOP_ACQ FSM_START_ACQ
  • FSM_START_ACQ [write-only]: State machine acquisition_start command (ignore on read)
    1: ACQ_START (start acquisition, only when FSM is idle)
  • FSM_STOP_ACQ [write-only]: State machine stop command (ignore on read)
    1: ACQ_STOP (stop acquisition, anytime)
  • RESERVED1 [read/write]: Reserved1
    Ignore on read, write with 0's
  • FSM_ACQ_NOW [read/write]: Acquire data immediately and don't wait for any trigger (ignore on read)
    1: Acquire immediately
    0: Wait for trigger
  • RESERVED2 [read/write]: Reserved2
    Ignore on read, write with 0's

HW prefix: acq_core_sta
HW address: 0x1
C prefix: STA
C offset: 0x4

RESERVED2[5:0] FC_FULL FC_TRANS_DONE
7 6 5 4 3 2 1 0
RESERVED1[3:0] FSM_ACQ_DONE FSM_STATE[2:0]
  • FSM_STATE [read-only]: State machine status
    States:
    0: illegal
    1: IDLE
    2: PRE_TRIG
    3: WAIT_TRIG
    4: POST_TRIG
    5: DECR_SHOT
    6: illegal
    7: illegal
  • FSM_ACQ_DONE [read-only]: FSM acquisition status
    Values:
    0: in progress
    1: completed
  • RESERVED1 [read-only]: Reserved
    Ignore on read, write with 0's
  • FC_TRANS_DONE [read-only]: External flow control transfer status
    Values:
    0: in progress
    1: completed
  • FC_FULL [read-only]: External flow control FIFO full status
    Values:
    0: not full
    1: full (data may be lost)
  • RESERVED2 [read-only]: Reserved
    Ignore on read, write with 0's
  • DDR3_TRANS_DONE [read-only]: DDR3 transfer status
    Values:
    0: in progress
    1: completed
  • RESERVED3 [read-only]: Reserved
    Ignore on read, write with 0's

3.3. Trigger configuration

HW prefix: acq_core_trig_cfg
HW address: 0x2
C prefix: TRIG_CFG
C offset: 0x8
31 30 29 28 27 26 25 24
INT_TRIG_THRES[15:8]
23 22 21 20 19 18 17 16
INT_TRIG_THRES[7:0]
15 14 13 12 11 10 9 8
RESERVED[9:2]
7 6 5 4 3 2 1 0
RESERVED[1:0] INT_TRIG_SEL[1:0] SW_TRIG_EN HW_TRIG_EN HW_TRIG_POL HW_TRIG_SEL
  • HW_TRIG_SEL [read/write]: Hardware trigger selection
    0: internal (data threshold)
    1: external (front panel trigger input)
  • HW_TRIG_POL [read/write]: Hardware trigger polarity
    0: positive edge/slope
    1: negative edge/slope
  • HW_TRIG_EN [read/write]: Hardware trigger enable
    0: disable
    1: enable
  • SW_TRIG_EN [read/write]: Software trigger enable
    0: disable
    1: enable
  • INT_TRIG_SEL [read/write]: Channel selection for internal trigger
    00: channel 1
    01: channel 2
    10: channel 3
    11: channel 4
  • RESERVED [read/write]: Reserved
    Ignore on read, write with 0's
  • INT_TRIG_THRES [read/write]: Threshold for internal trigger
    Treated as binary two's complement and compared to raw ADC data
